# Interpreter

PHP와 CodeMirror를 활용하여 구축한 웹 기반 온라인 컴파일러 및 코딩 교육(LMS) 플랫폼입니다. 
별도의 IDE 프로그램 설치 없이 브라우저 환경에서 코드를 작성하고 실행할 수 있으며, 학생들의 과제 제출, 강사의 코드 피드백, 학습 커뮤니티 기능을 통합적으로 제공합니다.

## 주요 기능

### 1. 웹 기반 온라인 컴파일러 및 실행 환경
- **코드 컴파일 및 실행**: 브라우저에서 작성된 코드를 PHP 백엔드(system, shell_exec 함수 등)로 전송하여 서버 단에서 컴파일 및 실행 결과를 반환합니다. (예: GCC 연동을 통한 C언어 컴파일 지원)
- **강력한 코드 에디터**: CodeMirror 라이브러리를 적용하여 수십 가지 프로그래밍 언어에 대한 구문 강조(Syntax Highlighting)를 지원합니다.
- **에디터 편의 기능**: 코드 자동 완성(Hint), 괄호 및 태그 자동 닫기, 코드 폴딩(Folding) 기능을 제공합니다.
- **사용자 맞춤형 UI**: 다양한 에디터 테마를 지원하며, 화면 분할 기능과 창 크기 조절(resize.js) 기능을 통해 유연한 코딩 환경을 구성할 수 있습니다.

### 2. 코딩 학습 및 과제 관리 시스템 (LMS)
- **과제 출제 및 관리**: 강사가 코딩 과제를 출제하고 기간 및 상세 내용을 관리할 수 있는 전용 게시판 환경을 제공합니다. (board.php, subject.php)
- **코드 제출 및 피드백**: 학생은 에디터에서 작성한 소스 코드를 즉시 서버로 제출할 수 있으며, 강사는 제출된 코드에 대해 평가와 피드백을 남길 수 있습니다. (submission.php, article_feedback.php)
- **코드 연습장**: 과제 제출 목적이 아닌, 알고리즘 문제 풀이나 개인적인 코드 테스트를 자유롭게 진행할 수 있는 독립된 컴파일 환경을 제공합니다. (practice_board.php)

### 3. 커뮤니티 및 게시판 시스템
- **자유게시판**: 개발 관련 질의응답 및 정보 공유를 위한 커뮤니티 공간을 지원합니다. (free_board.php)
- **자료실 기능**: 소스 코드 파일이나 강의 자료 등을 업로드하고 다운로드할 수 있는 파일 첨부형 게시판을 제공합니다. (fileboard_list.php, fileupload.php)
- **댓글 시스템**: 각 게시물 하단에 즉각적으로 소통할 수 있는 댓글 기능을 구현하였습니다. (free_board_reply.php)

### 4. 확장 기능 및 인프라
- **사용자 인증 및 권한 관리**: 회원가입 및 세션 기반 로그인을 통해 학생과 강사 등의 역할을 구분하여 시스템을 운영합니다.
- **DB 관리 도구 내장**: phpMyAdmin을 프로젝트 내부에 통합하여 관리자의 직관적인 데이터베이스 제어 및 테이블 관리를 지원합니다.
- **외부 API 연동 구조**: 안드로이드 등 모바일 애플리케이션과의 데이터 통신을 테스트하기 위한 API 엔드포인트가 포함되어 있습니다. (android/test.php)

## 기술 스택
- **Frontend**: HTML5, CSS3, JavaScript (jQuery), CodeMirror
- **Backend**: PHP
- **Database**: MySQL / MariaDB

## 상세 디렉토리 구조
```text
Interpreter/
├── html/
│   ├── android/             # 모바일/외부 클라이언트 통신용 API 스크립트
│   ├── board/               # LMS 및 커뮤니티 비즈니스 로직 최상위 폴더
│   │   ├── css/ & js/       # 게시판 전용 UI 레이아웃 및 스크립트
│   │   ├── upload/          # 파일 업로드 디렉토리 및 StopWatch 모듈
│   │   │   └── Downloads/   # 시스템 기초 테이블 세팅용 SQL 스키마 
│   │   ├── subject.php      # 과제 출제 모듈
│   │   ├── submission.php   # 과제 제출 및 DB 인서트 모듈
│   │   ├── comp.php         # 메인 컴파일 및 실행 모듈
│   │   └── make.php         # 소스코드 파일 생성 모듈
│   ├── compiler/            # CodeMirror 구문 강조, 테마, 자동완성 애드온
│   ├── css/ & js/           # 글로벌 레이아웃, 에디터 리사이징(resize.js) 기능
│   ├── phpadmin/            # DB 통합 관리를 위한 내장 phpMyAdmin
│   └── signupChk.php        # 회원가입 및 DB 유효성 검증 로직
```
## 설치 및 실행 방법

1. 리포지토리를 웹 서버 환경(예: Apache, Nginx의 htdocs 또는 /var/www/html)으로 클론합니다.
2. MySQL 또는 MariaDB 데이터베이스를 생성합니다.
3. `html/board/upload/Downloads/` 경로에 포함된 SQL 파일(`board.sql`, `login.sql`, `member.sql`, `f_submission.sql`)을 데이터베이스에 Import 하여 기초 테이블을 생성합니다.
4. PHP 소스 코드 내 데이터베이스 연결 부분(mysqli_connect 함수 등)의 호스트, 계정명, 비밀번호, 데이터베이스 이름을 서버 환경에 맞게 수정합니다.
5. 컴파일 기능 정상 작동을 위해 서버에 GCC 등 타겟 언어의 컴파일러가 설치되어 있는지 확인하고, PHP의 `system` 또는 `shell_exec` 함수 실행 권한이 허용되어 있는지 점검합니다.
6. 웹 브라우저를 통해 호스팅 주소로 접속하여 회원가입 후 기능을 이용합니다.
